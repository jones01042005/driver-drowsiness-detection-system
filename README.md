🚗 Driver Drowsiness Detection System

📌 Overview
An IoT-based safety system that detects driver fatigue using an eye blink sensor and provides real-time alerts to prevent accidents.

🚀 Features
👁️ Real-time eye blink monitoring
⚡ Fatigue detection using IR sensor
🔔 Instant alerts (buzzer)
🔌 Automatic control using relay (motor stop simulation)
💡 Low-cost and portable design

🏗️ Tech Stack
Arduino Nano
Eye Blink Sensor (IR-based)
Arduino IDE (C/C++)
Relay Module, Buzzer, DC Motor

⚙️ How It Works
Sensor tracks eye blink patterns
Arduino processes blink data
Detects abnormal/drowsy behavior
Triggers buzzer alert
Stops motor via relay (safety action)

🌍 Applications
Driver safety systems
Industrial worker monitoring
Fatigue detection systems

🔮 Future Improvements
Camera-based detection (Computer Vision)
Mobile alert system
Integration with vehicle systems

⭐ Key Highlight
A real-time IoT safety solution designed to reduce accidents caused by driver fatigue.
